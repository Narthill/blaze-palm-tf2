# Implement MediaPipe BlazePalm Model

Implement BlazePalm ( hand detector ) using Tensorflow 2.0.

## Usage

- Install dependencies

    ```
    pip install -r requirements.txt
    ```

- Training

    Check `train.ipynb` for training a model
